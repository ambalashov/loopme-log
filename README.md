# Introduction to loopme-log

Simple wrap for log4j2

## Deploy

Set environment variables for loopme s3 basket
LEIN_USERNAME
LEIN_PASSPHRASE

## Usage

```clojure
:plugins [[s3-wagon-private "1.1.2"]] 
:dependencies [[loopme/log "0.1.1"]]
```
